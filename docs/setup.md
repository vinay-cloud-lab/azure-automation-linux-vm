# Setup Guide

## 1. Create Azure Automation Account

- Go to Azure Portal
- Create Automation Account

## 2. Create Runbooks

- Add Python Runbooks
- Paste scripts from runbooks folder

## 3. Setup Hybrid Worker

- Install Hybrid Worker on Linux VM
- Register with Automation Account

## 4. Run Runbooks

- Select Hybrid Worker Group
- Execute scripts

## 5. Verify

- Login to VM
- Check users and services
