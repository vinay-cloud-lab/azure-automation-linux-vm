# Azure Automation - Linux VM Management

This project demonstrates how to automate common Linux VM tasks using Azure Automation and Hybrid Workers.

## 🚀 Overview

Using Azure Automation, I automated tasks across multiple Linux virtual machines without logging into each system manually.

## 🔧 Features

- User creation and deletion
- Service monitoring (NGINX)
- Software installation (AnyDesk)
- Execution across multiple VMs using Hybrid Worker

## 🧠 How it works

- Runbooks are created using Python
- Hybrid Worker is installed on Linux VMs
- Runbooks execute locally on each VM
- Same script runs across multiple machines
